Project Description
-------------------

1) Obstacle Climbing
2) Zigbee Control
3) Different forms of Hexapod gait
4) Hexapod dancing
   
Software required
-----------------

+    ImageCraft IDE for ICC AVR 7
+    AVR Studio

Build
-----------------

Open the project in ICC AVR 7
Do Project->Rebuild All to generate the hex file
Dont do make project

Make a dummy project in AVR Studio
Burn the HEX file using the dummy project on the hexapod
